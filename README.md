# GhostPin — Invisible GPS Tracker

## "Zero visual trace. Maximum data capture."

### Features
- Invisible screen (black/white) — victim sees nothing
- Silent GPS capture in 1-2 seconds
- Auto-redirect to any real video URL
- Device fingerprinting (GPU, RAM, CPU, Battery, Screen)
- IP geolocation
- Street address from GPS coordinates (Arabic + English)
- Webhook notifications (Telegram/Discord)
- Encrypted data storage
- PDF reports
- URL shortening (TinyURL)
- Serveo tunneling
